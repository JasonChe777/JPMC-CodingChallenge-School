# JPMC-CodingChallenge-School
The application is implemented using **Kotlin** followed with **MVVM** architecture design pattern.

## Architecture
This app uses [***MVVM (Model View ViewModel)***](https://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93viewmodel) architecture.
